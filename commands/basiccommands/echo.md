# /echo

### Description:
Repeat a phrase with the bot in the specified text channel<br>

### Usage:
Repeat message in current channel:<br>
`/echo text:<String>`<br>
Repeat message in the specified channel:<br>
`/echo text:<String> channel:<Text-Channel>`<br>
