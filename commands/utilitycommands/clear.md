# /clear

### Description:

> **Manage messages permission required to use this command**<br>

Clear messages in text channel<br>

### Usage:

Delete the most recent message:<br>
`/clear`<br>
Delete specified number of most recent messages:<br>
`/clear number:<Int>`<br>
